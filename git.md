### Staging Area
`git status` 

#### Para añadir un solo archivo
`git add nombre_del_archivo` 
 
#### Para añadir todo lo que hay en el directorio actual
`git add --all` 

#### o bien
`git add .` 

#### Commit
`git commit -m 'Primer commit'` 

#### Etiquetas
